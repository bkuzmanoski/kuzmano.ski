This file is MDX. It renders through the shared component map, so headings,
links and code blocks pick up the classic system styling automatically.

Code is highlighted at build time by Shiki — there is no highlighter in the
client bundle:

```ts
const greeting: string = "Hello from a prerendered page";
console.log(greeting);
```

Because it is MDX rather than plain Markdown, an interactive component can be
dropped straight into the prose whenever a piece of writing needs one.
